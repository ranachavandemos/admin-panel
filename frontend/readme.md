#  Admin Panel Frontend

React-based frontend built with **Vite**, **Tailwind CSS**, and **React Router**, designed to integrate with the Admin Panel backend.

---

## Tech Stack
- **Framework:** React (v18+)
- **Build Tool:** Vite
- **Styling:** Tailwind CSS + PostCSS + Autoprefixer
- **Routing:** React Router DOM
- **HTTP Client:** Axios

---

## Installation

### Prerequisites
- Node.js **v18+**
- npm **v9+**

### Setup
```bash
cd admin-panel-frontend
npm install
