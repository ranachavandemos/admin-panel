#  Admin Panel Backend

Fastify + TypeScript backend for Admin Panel with JWT authentication, MySQL database, and file upload handling.

---

## Tech Stack
- **Runtime:** Node.js (v18+)
- **Framework:** Fastify
- **Language:** TypeScript
- **ORM:** Sequelize
- **Database:** MySQL
- **Auth:** JWT + bcrypt
- **File Uploads:** fastify-multer + @fastify/multipart

---

## Installation

###  Prerequisites
- Node.js **v18+**
- npm **v9+**
- MySQL server

###  Setup
```bash
cd admin-panel-backend

Install:
```
npm install
```

Run:
```
npm run dev
```
